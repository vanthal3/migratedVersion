Specify the path to the Jtl report files, relative to the workspace root.

  • You can specify multiple files and directories separated by semicolon.
    * Jmeter writes summariser statistics to jmeter.log. To enable logging summariser statistics to separate log file add the property to jmeter.properties file to #Logging Configuration block : "log_file.jmeter.reporters.Summariser=filename.log"
